# Dashboard UI Kit — RFDD

Power BI–style management dashboard styled in the RFDD editorial-financial language. RFDD's "Demos" pages embed live Power BI reports we cannot inspect — this kit is a stylistic interpretation, not a clone.

## Components
- `DashHeader.jsx` — slim header with logo, dashboard title, period selector
- `FilterBar.jsx` — segmented period + region filters
- `KpiTile.jsx` — KPI tile with display-serif value + variance pill
- `BarChart.jsx`, `LineChart.jsx`, `Donut.jsx` — minimal SVG charts in the brand palette
- `DataTable.jsx` — financial table with right-aligned numerics

## Caveats
- Real RFDD dashboards are Power BI; this is an HTML interpretation only.
