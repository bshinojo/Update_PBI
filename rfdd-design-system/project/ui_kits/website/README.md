# Website UI Kit — RFDD

A pixel-spirit recreation of **rfdd.com.ar** in the language of the design system. The original site is built on a generic WordPress theme + Revolution Slider — this kit reinterprets its IA and copy through the brand's editorial-financial visual system rather than cloning the WP chrome.

## Components

- `SiteHeader.jsx` — sticky transparent-on-hero / paper-on-scroll, logo + nav + login CTA
- `Hero.jsx` — editorial slider with eyebrow + serif promise + lead + dual CTA
- `Differentiators.jsx` — six tracked-caps differentiators band
- `ServicesGrid.jsx` — three service-pillar cards with top-rule
- `ClientsWall.jsx` — placeholder client wordmark grid
- `FAQ.jsx` — accordion FAQ with serif questions, sans answers
- `SiteFooter.jsx` — full-width navy with paper text, mono dark logo
- `Buttons.jsx` — `<Button variant>` shared primitive

## How to run
Open `index.html` — everything is inline JSX/Babel.

## Caveats
- Hero photography is a placeholder navy gradient; real site uses stock corporate imagery.
- Client logos are SVG wordmark placeholders.
